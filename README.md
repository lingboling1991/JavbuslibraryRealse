## 帮助

- 使用新的中文字幕推送功能，需要你将APP加入白名单（2.5+）
- 点击剧照标题可临时更换图片源（2.5+）
- 遇到网络超时无法访问，点击重试或者尝试更换网络环境
- 长按左上角返回按钮可以快速回到首页（2.2+）
- 长按详情标题可以翻译中文（2.1+）
- 长按“我的收藏”影片可删除影片（2.1+）
- 长按详情识别码可以查看该识别码相关的影片（2.0+）
- “管理我的关注”中滑动可删除列表项目（2.0+）

因为众所周知的原因，可能会被列成危险和病毒APP
